ファイルをAESで暗号化するためのソースです
64bit版Windows7でのみ動作確認しています

※使い方
1.暗号化
  ./aes.exe -e "暗号化したいファイル名" "暗号化した後のファイル名"
  パスワードの入力

  注意
    暗号化したいファイル名はテキスト形式のファイルでお願いします
    -eは第一引数でお願いします
    出力ファイルはバイナリ形式になります

2.復号
  ./aes.exe -d "暗号ファイル"
  パスワードの入力

  注意
    -dは第一引数でお願いします
    結果は標準出力に表示されます
